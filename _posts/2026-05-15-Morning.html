Summary: Ollama has expanded its developer ecosystem with the introduction of support for the Codex app, aiming to streamline the coding and review process (ollama). Users can access this integration by updating to Ollama version 0.24 and utilizing the `ollama launch codex-app` command (ollama). The Codex integration introduces several productivity-focused features designed to keep developers within a single workspace. Users can now review code and iterate via comments without leaving the environment (ollama). Additionally, the app includes a built-in browser capable of loading local servers and sites, allowing developers to visually edit websites by annotating pages and applying changes through chat (ollama). For those wishing to return to a previous state, the `--restore` flag can be used to revert to an existing Codex experience (ollama). To optimize performance within the app, Ollama has suggested several models for users to try (ollama). These include vision-supported models like kimi-k2.6:cloud and glm-5.1:cloud (ollama). For those without a paid Ollama cloud subscription, the company recommends models known for reliable tool calling, such as qwen3.6, gemma4:31b:cloud, and nemotron-3-super:cloud (ollama).
